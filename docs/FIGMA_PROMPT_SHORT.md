## Project: StyleMate - AI Fashion Stylist App

**Platform:** React Native (iOS & Android)  
**Target:** Gen Z fashion enthusiasts (18-28)  
**Goal:** Modern, intuitive UI/UX for AI-powered fashion styling

---

## Design System

**Colors:**
- Primary: `#6366F1` (Indigo)
- Secondary: `#8B5CF6` (Purple)
- Background: `#0F0F23` (Dark blue-black)
- Text: `#FFFFFF` (White)
- Pinterest: `#E60023`

**Style:** Dark theme, glassmorphism, modern minimalism, Gen Z appeal

---

## Key Screens to Design

### 1. **Home Screen**
- Personalized greeting + weather widget
- 4 action cards: "Ask AI Stylist" (primary), "Style Check", "Find Similar Items", "Enhance with Pinterest" (optional)
- Stats section: Wardrobe count, Store items

### 2. **AI Stylist Screen** (Main Feature)
- Header: Title + optional Pinterest badge + stats
- Context chips: Occasion (casual/professional/date/party) + Weather (cold/mild/warm/hot)
- Large swipeable outfit card (Tinder-style)
- Progress indicator + swipe instructions
- Undo banner after swipes

### 3. **Wardrobe Screen**
- Header with search/camera/gallery/selection icons
- Category tabs (All, Tops, Bottoms, Shoes, etc.)
- 2-column grid of clothing items
- Selection mode bar when active

### 4. **Pinterest Style Screen** (Optional Feature)
- Header emphasizing "optional" nature
- Pinterest board analyzer input
- Analyzed boards list (if any)
- "How It Works" + "Benefits" sections
- Empty state encouraging but not pushy

### 5. **Profile Screen**
- User avatar + info
- Stats grid
- Preferences, Achievements, Settings sections

### 6. **Style Check Screen**
- Camera interface
- AI analysis results display

---

## UX Principles

1. **Instant Gratification** - No long loading, immediate feedback
2. **Clarity** - One primary action per screen
3. **Delight** - Smooth animations, micro-interactions
4. **Optional Features** - Pinterest clearly optional, not required
5. **Accessibility** - WCAG AA, 44px touch targets

---

## Key Requirements

- Dark theme with glassmorphism effects
- Smooth 60fps animations
- Clear visual hierarchy
- Pinterest integration feels optional, not required
- Mobile-first, responsive design
- System fonts (SF Pro iOS, Roboto Android)

---

## Deliverables

- Complete screen designs
- Design system (colors, typography, components)
- Component library
- Empty/loading/error states
- Animation guidelines
- Responsive variations
